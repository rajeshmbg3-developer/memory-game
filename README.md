## Installation

Install the game with git

```bash
    git clone https://github.com/rajeshmbg3-developer/memory-game.git
    cd memory-game
```
## Running Game Locally

Install dependencies:

```bash
    nmp install
```

Run application:

```bash
    npm run start
```
